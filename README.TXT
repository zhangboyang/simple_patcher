simple patcher

一个简易的用来打补丁的程序

程序说明：
    make_script : 根据两个等长文件的差异生成补丁脚本
    run_script  : 按照脚本指令修改文件

脚本格式：
    脚本的解析方式为按空白符分割指令，因此指令中不得有空白符

    注释：
        单行注释：一行中 # ; // 及它们后面的字符会被忽略
        块注释：所有 /* 与 */ 之间的内容将被忽略

    输入输出：  
        读入指令：指令格式为 >[文件名]，将所指定的文件读入数据缓冲区
        写出指令：指令格式为 <[文件名]，将数据缓冲区写入所指定文件

    数据校验：
        MD5 校验指令：指令格式为 ?[MD5 值]，将数据缓冲区中的数据的 MD5 值与所给定 MD5 值比较

    指针移动指令：
        一共有3种格式的指令：
            :A/B   将指针移到 A 处，并计算偏移量 D = A - B
            :/B    将指针移到 B + D 处
            :A     将指针移到 A 处

    数据操作指令：
        所有其他指令均视为数据操作指令，格式为 XY，其中 X 与 Y 为一位十六进制数字
        该指令会把当前指针所指位置的字节置为 0xXY 然后将指针后移一位

使用方法：
    make_script [旧文件] [新文件]
	生成从旧文件到新文件的补丁脚本，并写到标准输出

    run_script
        从标准输入读取脚本并运行

    run_script [脚本文件]
        运行指定的脚本文件

编译方法：
    本程序使用了 stricmp 以及 cryptdll.dll 中的 MD5 函数，因此只能在 Windows 系统上运行
    如果已经设置好 gcc 的环境变量，可以直接双击 compile.bat 执行编译。
    手动编译命令为：
        gcc -O2 -g -Wall -o make_script.exe make_script.c md5.c
        gcc -O2 -g -Wall -o run_script.exe run_script.c md5.c
